## アイデア帳 (Ideas Backlog)

> まだ開発に着手していない、今後作りたいアプリのネタ

### IDEA-001: バーチャル共同作業スペース

| 項目 | 内容 |
|------|------|
| **仮称** | もくもくメイト / Mokumoku Mate |
| **着想日** | 2026-03-08 |
| **課題** | 一人で休日作業しているとき、モチベーションが続かない。孤独感で集中が切れる。頑張りを誰かに認めてほしい |
| **コンセプト** | 「カフェの隣の席」-- 顔は見えないが、誰かが隣で頑張っているのがわかる。自分の頑張りも認めてもらえる |
| **ターゲット** | 個人開発者、フリーランス、休日の自主学習者（一人だと頑張れない人） |
| **優先度** | 未定 |
| **ステータス** | アイデア → 検討中 |

#### 設計メモ

**作業証明（"いるフリ"排除）**
- スマホのインカメラで人体検知（MediaPipe等）。映像は送らず「人がいる/いない」の1bit情報のみ送信
- PC作業者はキーボード/マウス活動検知も併用（デスクトップアプリ: Tauri or Electron）
- 読書・手書き作業の人もスマホカメラで在席検知可能。追加デバイス不要

**プロフィール（選べる自己開示）**
- ニックネーム + 自作アバター/アイコン（顔写真不要・本名不要）
- 作業内容は任意記入（「読書中」「開発中」「英語の勉強」等）
- 完全匿名ではない。「自分の頑張りが自分のもの」として見える

**頑張りの可視化（自己顕示欲を満たす）**
- 作業時間の累積表示（今日/今週/今月）
- 連続作業日数ストリーク
- 実績バッジ（「早起き作業」「深夜の努力家」「100時間達成」等）
- 作業時間ランキング（週間/月間）
- SNSシェア機能（「今日5時間やりました」）

**コミュニケーション設計（集中を妨げない）**
- DM・チャット機能は**入れない**（会話が目的になり作業が止まる。チャットしたいならDiscord/LINEを使えばいい）
- ワンタップスタンプ（「頑張ってるね」「お疲れ」等のリアクション）
- 作業宣言つぶやき（開始時「今日は○○やる」/ 終了時「○時間やりきった！」）→ 作業中は投稿不可
- ポモドーロ休憩中のみ投稿可 → 集中中は書けないルール

**あえて入れないもの**
- 顔出しビデオ → 身だしなみが気になり集中できない
- フリーチャット・DM → コミュニケーションアプリではない。出会い目的排除
- フリータイムライン → Twitterになってしまう

**技術候補**
- Tauri / Electron（デスクトップ、OS入力検知）+ React Native（スマホ、カメラ検知）
- Supabase Realtime（プレゼンス共有）
- MediaPipe（デバイス上で人体検知→映像はサーバーに送らない）

## 新しいアイデアの追加方法

新しいアイデアを追加する際は、以下のテンプレートを使ってください：

```markdown
### IDEA-XXX: タイトル

| 項目 | 内容 |
|------|------|
| **仮称** | アプリ名 |
| **着想日** | YYYY-MM-DD |
| **課題** | どんな問題を解決するか |
| **コンセプト** | どうやって解決するか |
| **想定機能** | 主な機能リスト |
| **ターゲット** | 誰向けか |
| **差別化** | 既存サービスとの違い |
| **技術候補** | 使いそうな技術 |
| **優先度** | 高/中/低/未定 |
| **ステータス** | アイデア → 検討中 → 設計中 → 開発中 |
```
