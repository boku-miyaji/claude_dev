## アイデア帳 (Ideas Backlog)

> まだ開発に着手していない、今後作りたいアプリのネタ

### IDEA-001: バーチャル共同作業スペース

| 項目 | 内容 |
|------|------|
| **仮称** | もくもくメイト / Mokumoku Mate |
| **着想日** | 2026-03-08 |
| **課題** | 一人で休日作業しているとき、モチベーションが続かない。孤独感で集中が切れる |
| **コンセプト** | 画面越しでも「一緒に頑張っている人がどこかにいる」ことがわかるだけで頑張れる。少しの"見張られている感"がサボり防止になる |
| **想定機能** | |
| | - 作業中の人数・状態をリアルタイム表示（「今5人が作業中」） |
| | - 作業タイマー（ポモドーロ式）の共有・同期 |
| | - 軽い存在感の演出（アバター、タイピング音、進捗バー等） |
| | - 「頑張ってるね」スタンプ等の軽量コミュニケーション |
| | - 作業ログ・振り返り（今日どれだけ頑張ったか） |
| **ターゲット** | 個人開発者、フリーランス、休日の自主学習者 |
| **差別化** | Discordのもくもく会やFocusmateとは違い、カメラOFF・テキスト不要。"存在だけを共有"する超軽量アプローチ |
| **技術候補** | WebSocket / Supabase Realtime, React / React Native, 匿名セッション |
| **優先度** | 未定 |
| **ステータス** | アイデア |

## 新しいアイデアの追加方法

新しいアイデアを追加する際は、以下のテンプレートを使ってください：

```markdown
### IDEA-XXX: タイトル

| 項目 | 内容 |
|------|------|
| **仮称** | アプリ名 |
| **着想日** | YYYY-MM-DD |
| **課題** | どんな問題を解決するか |
| **コンセプト** | どうやって解決するか |
| **想定機能** | 主な機能リスト |
| **ターゲット** | 誰向けか |
| **差別化** | 既存サービスとの違い |
| **技術候補** | 使いそうな技術 |
| **優先度** | 高/中/低/未定 |
| **ステータス** | アイデア → 検討中 → 設計中 → 開発中 |
```
