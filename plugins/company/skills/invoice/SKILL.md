## サブコマンド

### `/invoice` （引数なし）
今月のサマリーを表示:
- 売上合計、経費合計、粗利
- 案件別の稼働時間と実質時給
- 未入金の請求書一覧
- 次の税金支払い予定

### `/invoice add <ファイルパス>`
請求書ファイル（PDF/画像）を解析して登録:
1. Read ツールでファイルを読み取る
2. 請求書番号、金額、日付、請求先を抽出
3. 登録済み projects から Claude が案件を推定（確信度が低ければユーザーに質問）
4. Supabase `invoices` テーブルに POST

### `/invoice project`
案件一覧を表示。

### `/invoice project add "案件名" --client "会社名"`
案件を新規登録。オプション:
- `--type project|monthly|hourly|consulting`
- `--rate 5000` （目標時給）
- `--budget 1000000`

### `/invoice expense "内容" 金額`
経費を登録。
- カテゴリを Claude が自動判定
- `--date 2026-03-15` で日付指定（省略時は今日）
- `--receipt path/to/receipt.jpg` でレシート添付

### `/invoice sync`
Google Calendar から稼働時間を同期:
1. google-calendar MCP で今月のイベントを取得
2. `[仕事]` プレフィックスのイベントだけ抽出
3. イベント形式: `[仕事] {会社名} {内容}`
4. 会社名と内容から、登録済み projects を Claude が分析してマッチ
5. マッチ確信度が低い場合はユーザーに質問:
   「"[仕事] B社 MTG" はどの案件に紐づけますか？ 1. B社コンサル 2. 新規案件として登録」
6. calendar_event_id で重複チェック済みのイベントはスキップ
7. 稼働時間を `time_entries` に登録

### `/invoice tax`
税金シミュレーション:
1. 今年度の売上合計・経費合計を集計
2. 課税所得 = 売上 - 経費 - 青色申告特別控除(65万)
3. 以下を計算:

| 税目 | 計算方法 |
|------|----------|
| **所得税** | 累進課税（5%〜45%）+ 復興特別所得税 2.1% |
| **住民税** | 課税所得 × 10% + 均等割 5,000円 |
| **個人事業税** | (課税所得 - 290万) × 5%（290万以下なら0） |
| **消費税** | 売上の消費税 - 仕入控除（課税売上1,000万超の場合） |

4. 納付スケジュールを表示:
```
📅 2026年度 納付スケジュール
┌──────────┬──────────┬──────────┐
│ 納付期限   │ 税目     │ 概算額    │
├──────────┼──────────┼──────────┤
│ 2027/03/15 │ 所得税   │ ¥xxx,xxx │
│ 2027/03/31 │ 消費税   │ ¥xxx,xxx │
│ 2027/06    │ 住民税1期 │ ¥xx,xxx  │
│ 2027/07    │ 予定納税1期│ ¥xxx,xxx │
│ ...        │ ...      │ ...      │
└──────────┴──────────┴──────────┘
```

## 案件マッチングのルール

Calendar イベント `[仕事] {会社名} {内容}` から案件を推定する際:

1. `projects` テーブルから `status = 'active'` の案件一覧を取得
2. 会社名を `client_name` と照合（完全一致でなくてよい、Claude が判断）
3. 内容が案件の `description` や `name` と関連するか分析
4. **確信度が高い**: 自動で紐付け、結果を報告
5. **確信度が低い or 候補が複数**: ユーザーに質問して確定
6. **どの案件にも該当しない**: 新規案件として登録するか質問

## 税金計算の詳細（個人事業主・青色申告）

### 所得税率テーブル
| 課税所得 | 税率 | 控除額 |
|---------|------|--------|
| 〜195万 | 5% | 0 |
| 〜330万 | 10% | 97,500 |
| 〜695万 | 20% | 427,500 |
| 〜900万 | 23% | 636,000 |
| 〜1,800万 | 33% | 1,536,000 |
| 〜4,000万 | 40% | 2,796,000 |
| 4,000万超 | 45% | 4,796,000 |

### 計算フロー
```
売上（税込） → 消費税を分離 → 事業収入
事業収入 - 必要経費 = 事業所得
事業所得 - 青色申告特別控除(65万) = 課税所得ベース
課税所得ベース - 基礎控除(48万) - その他控除 = 課税所得
→ 所得税 = 課税所得 × 税率 - 控除額
→ 復興特別所得税 = 所得税 × 2.1%
→ 住民税 = 課税所得 × 10% + 均等割
→ 事業税 = (事業所得 - 290万) × 5%
```
