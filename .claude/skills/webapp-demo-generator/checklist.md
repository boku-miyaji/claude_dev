# Demo Generation Checklist

## 対象確認

- [ ] アプリURL
- [ ] ログイン方式（SSO / ID-PW / トークン / バイパス可）
- [ ] 実行環境（ローカル / ステージング / 本番相当）
- [ ] テストアカウント（有無、作成方法）
- [ ] テストデータ（有無、リセット方法）
- [ ] 対象ブラウザ（Chromium / Firefox / WebKit）

## デモ目的

- [ ] 誰向けか（経営層 / 現場ユーザー / 開発者 / 投資家）
- [ ] 何を伝えたいか（業務価値 / 操作性 / AI機能 / 差別化）
- [ ] 何分以内に収めるか（推奨: 3-5分 / 最大: 10分）
- [ ] ナレーション有無
- [ ] 字幕有無
- [ ] BGM有無

## デモシナリオ

- [ ] 操作ステップの一覧
- [ ] 各ステップの所要時間
- [ ] 強調したい画面・操作
- [ ] 待機が必要な箇所（AI生成、ローディング等）
- [ ] データの前提条件（事前に投入すべきデータ）

## 実装条件

- [ ] Node.js バージョン
- [ ] Playwright 導入可否
- [ ] ffmpeg 利用可否
- [ ] TTS 利用可否（OpenAI TTS / Google TTS / VOICEVOX 等）
- [ ] CI 利用可否（GitHub Actions / GitLab CI 等）
- [ ] Docker 利用可否

## 出力要件

- [ ] 動画フォーマット（mp4 / webm）
- [ ] 解像度（1440x900 / 1920x1080 / カスタム）
- [ ] フレームレート（30fps / 60fps）
- [ ] 字幕フォーマット（srt / vtt / 焼き込み）
- [ ] 出力先（ローカル / S3 / GitHub Release）

## セキュリティ

- [ ] 個人情報の表示可否
- [ ] マスキング要否（名前、メール、金額等）
- [ ] シークレット管理方法（.env / GitHub Secrets / Vault）
- [ ] 録画データの保存ポリシー
- [ ] 社外公開可否
