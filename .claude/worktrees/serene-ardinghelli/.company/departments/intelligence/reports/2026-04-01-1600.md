## 重要アラート

### [ALERT-1] Axios npm サプライチェーン攻撃 -- 北朝鮮関与の疑い、週1億DLパッケージが侵害 (2026-03-31)

Axios（週間1億DL超）の公式npmパッケージが侵害された。攻撃者がメンテナーのnpmアカウントを乗っ取り、バックドア入りの v1.14.1（latestタグ）と v0.30.4（legacyタグ）を公開。悪意のある依存関係 `plain-crypto-js@4.2.1` を注入し、postinstallフックでクロスプラットフォームRAT（Remote Access Trojan）をデプロイ。macOS/Windows/Linux全環境が対象。クラウドアクセスキー、DBパスワード、APIトークンを窃取する。

**影響**: 2026-03-31 00:21 ~ 03:29 UTC の間にaxiosをインストール/アップデートした場合、RATが仕込まれている可能性あり。

**対策**: 即座にaxiosバージョンを確認し、1.14.1 / 0.30.4 を使用している場合はダウングレードまたはアップデート。Socket.devが6分で自動検出。Google Cloud Blogによると北朝鮮関連のアクターが関与の疑い。

### [ALERT-2] Claude Code ソースコード流出 -- npm ソースマップから512,000行が露出 (2026-03-31)

Claude Code v2.1.88 のnpmパッケージに 59.8MB のソースマップファイルが誤って含まれ、512,000行（約1,900ファイル）のTypeScriptソースコードが完全に露出。`.npmignore` の設定漏れが原因。Anthropicは「リリースパッケージングの人的ミスであり、セキュリティ侵害ではない」と発表。

**注目**: 流出コードから未発表機能「KAIROS」（常時稼働型自律エージェントモード）の存在が判明。バックグラウンドでプロジェクトを監視し、夜間に「dreaming」（メモリ統合）を実行、プロアクティブにタスクを開始する機能。コンパイル時フラグでゲートされており外部リリースには含まれていない。

**二重のリスク**: この流出タイミングがAxios攻撃と重なっており、Claude Codeをnpm経由でインストールしたユーザーが同時にAxiosの侵害バージョンを取得した可能性あり。

### [ALERT-3] MCP Tool Poisoning Attack -- Invariant Labs が実証攻撃を公開 (2026-04-01)

Invariant LabsがMCPの「Tool Poisoning Attack」を公開。MCPサーバーのツール説明（description）に悪意のある隠し命令を埋め込むことで、AIエージェントが不正操作を実行する攻撃ベクター。概念実証ではClaude DesktopとCursorからSSH鍵と設定ファイルの窃取に成功。

**仕組み**: ユーザーには「Add Numbers」のような無害なツール名が表示されるが、AIモデルが受信するツール説明文にはデータ窃取やコマンド実行の隠し指示が含まれる。悪意のあるサーバーが信頼済みサーバーの動作をハイジャックすることも可能。

**対策**: サードパーティMCPサーバーの接続に注意。OWASP MCP Top 10も策定中。
