# ビジョン設計: Self-Focus Platform

**日付**: 2026-04-03
**起因**: diary → dashboard 統合検討の壁打ちから発展
**ステータス**: confirmed（社長承認済み）

## 上位ビジョン

> **「自分が幸せに生きるためのシステム」**

### 核心原則

1. **自分にフォーカス** — SNS的な他者比較とは一線を画す。自分の人生に集中する
2. **AIが一番の理解者** — 感情・夢・価値観・思考パターンを蓄積し、暗黙の意図を理解して動く
3. **良い未来へ導く** — 将来あるべき姿を提示し、そこへ引き上げる。しんどい時は励ます
4. **日常に溶け込む** — 仕事も感情も夢も全部ここにある。モバイルで断片メモ
5. **仕事は二の次** — 事業管理は機能の一部。自己理解・自己実現が最優先

### diary のビジョンとの関係

diary の「最強の相棒となるAIジャーナリングアプリ」は、このビジョンの部分集合:

```
Self-Focus Platform（上位概念）
├── 自己理解（感情・価値観・パターン分析）  ← diary のコア
├── 自己実現（夢・目標・成長記録）          ← diary のコア
├── 仕事・事業管理                          ← dashboard のコア
├── 能力向上・スキル                        ← 新要素
└── AI相棒（全てを統合理解）                ← 両方の進化形
```

## ポジショニング

| 軸 | このシステム | 競合（Motion, Reclaim等） | SNS |
|---|---|---|---|
| フォーカス | **自分だけ** | タスクだけ | 他者との比較 |
| AIの役割 | **人生の相棒** | スケジュール最適化 | なし |
| データの価値 | **自己理解が深まる** | 効率が上がる | 承認欲求 |
| 時間軸 | **過去→現在→未来** | 今日〜来週 | 今この瞬間 |
| 感情 | **核心** | 無視 | 消費 |

### マーケティング分析との接続

競合調査で発見した3つの独自ギャップに、4つ目を追加:

1. 事業コンテキストの深さ
2. 組織フィードバックループ
3. 永続的文脈管理
4. **経営者の全人格的理解（感情・夢・価値観込み）** ← NEW

## データ設計原則

### 蓄積するもの（= 自分を理解するための材料）

| データ | ソース | 何がわかるか |
|---|---|---|
| プロンプト（思考ログ） | Claude Code, AI Chat | 何を考えていたか、どんな意思決定をしたか |
| 感情・ジャーナル | diary入力、断片メモ | 感情パターン、価値観、心理状態 |
| 夢・目標 | 100の夢リスト、目標設定 | 何を大事にしているか、どこへ向かいたいか |
| 成果物・分析結果 | 部署の出力、レポート | 何を成し遂げたか |
| タスク・意思決定ログ | タスク管理、notes | 判断の履歴とその結果 |

### 蓄積しないもの

| データ | 理由 |
|---|---|
| AIの応答全文 | 自分にフォーカス。プロンプトから思考がわかる |
| 他者のSNS投稿 | 比較は不要 |
| 他者の評価・ランキング | 自分にフォーカス |

## AIの振る舞い設計

### 基本姿勢
- **理解者**: 毎回同じ指示をしなくても、裏に隠されたコンテキストを理解している
- **導き手**: 将来のあるべき姿を提示し、そこへ引き上げる
- **励まし**: しんどい時に寄り添い、前を向ける
- **記憶者**: 忘れがちなことをまとめて振り返れるようにする

### 自己分析（言語化の支援）
- diary・行動データ・プロンプトログから多角的に分析
- MBTI推定、Big5、強み・才能分析、仕事適合性、感情トリガーマップ、価値観の優先順位、行動リズム、ストレス耐性、成長軌跡、コミュニケーションスタイル
- 質問票ではなく **実際のデータから推定**（より正確）
- 「自分はこういう人間なのかも」の言語化を手伝う
- 定期的に再実行し、**自分の変化が見える**

### やらないこと
- 他者と比較しない
- 強制しない（Calm Technology）
- 音声入力（不要）

## 日常の接点

```
朝（起床〜通勤）
  └── モバイルで今日のフォーカス確認
  └── AIからの一言（励まし or 気づき）

日中（仕事）
  └── Claude Code でプロンプト自動記録
  └── 断片メモ（移動中にスマホで）
  └── タスク管理・PJ管理

夕方（振り返り）
  └── ジャーナリング（今日どうだった？）
  └── AI が感情を検出し、パターンに気づく

週末（俯瞰）
  └── 週次ナラティブ自動生成
  └── 夢・目標の進捗確認
  └── 成長の可視化
```
