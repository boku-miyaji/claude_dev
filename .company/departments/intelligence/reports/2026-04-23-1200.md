## 📄 注目論文（arXiv April 2026）

### **(2026-04-07) AgentGL: Towards Agentic Graph Learning with LLMs via Reinforcement Learning**

**arXiv**: https://arxiv.org/abs/2604.05846v1

**著者**: Yuanfu Sun, Kang Li, Dongzhe Fan, Jiajin Liu, Qiaoyu Tan

**受理**: ACL 2026 Main Conference

**要点**:

- **問題**: LLM agents が graph-structured data（ノード・エッジ・関係性）を扱う際、「unstructured text」として扱う傾向 → topology awareness 喪失
- **提案**: **Agentic Graph Learning (AGL)** — "interleaved topology-aware navigation + LLM-based inference"
  - Graph-native tools を agent に供給
  - RL feedback loop で multi-scale exploration を学習
- **成果**:
  - **Node classification**: +17.5% accuracy（benchmark datasets）
  - **Link prediction**: +28.4% improvement
  - Code 公開済み（reproducibility 確保）

**technical depth**:
→ Agent が「グラフのノード間の semantic distance」を推測して jump するのではなく、「neighborhood structure」を明示的に参照しながら navigation できるようになる。ナレッジグラフ、社会ネットワーク、recommendation system に即応用可能。

**focus-you への含意**:
→ **focus-you の diary + insight generation をグラフとして再建築**するチャンス。emotion cluster → narrative arc → growth event を「semantic graph」として表現し、**GAM-inspired graph learning** で過去ナラティブから新規 insight を derive する agent 設計が可能に。
