## 📄 注目論文（arXiv April 2026）

### **(2026-04-04～09) Memory Intelligence Agent (MIA) フレームワーク**

**arXiv**: https://arxiv.org/abs/2604.04503 (v3: 2026-04-09)

**著者**: 深度推論エージェント向けのメモリ統合フレームワーク

**要点**:
- **3層構成**: Memory Manager（非パラメータ）→ Planner（パラメータ、LLM agent） → Executor（検索・分析）
- **ビディレクショナル記憶進化**: パラメータ記憶と非パラメータ記憶の相互変換ループで、test-time learning 時に オンザフライで Planner 更新
- **強化学習ハイブリッド**: Planner と Executor 間の協力を交互強化で最適化
- **自己改善**: reflection + unsupervised judgment で推論・自己進化ブースト

**評価**: 11 ベンチマークで検証。既存の trajectory-retrieval 法の「メモリ進化の効率性悪化」「蓄積・検索コスト増加」を解決。

**focus-you への直結**:  
→ **HD の knowledge_base 進化ルール改善に直結**。現在の「昇格候補 confidence」メカニズムはスタティック。MIA の「パラメータ↔非パラメータ変換」を参考に、社長の学習ペースに応じた **動的なナレッジ昇格** を実装検討の価値あり。特に `knowledge_base` の confidence スコア更新時に、反射（reflection）ステップを挿入することで「なぜこれが重要か」の根拠を明示化できる。

---

### **(2026-04-14) GAM: Hierarchical Graph-based Agentic Memory**

**arXiv**: https://arxiv.org/abs/2604.12285 (2026-04-14)

**要点**:
- **記憶エンコーディング ≠ 統合**: 新情報取得と長期保持のバランス困難を、encoding と consolidation を **明示的に分離** することで解決
- **グラフ構造**: 進行中の dialogue を event progression graph に隔離 → semantic shift 時のみ topic network に統合
- **干渉最小化**: 短期 context と長期 memory の衝突を排除、consistency 保持

**評価**: LoCoMo・LongDialQA で state-of-the-art。reasoning accuracy + efficiency 両立。

**focus-you への示唆**:  
→ AI Partner の「daily reflection」と「長期ナラティブ」の分離に適用可能。現在のシステムでは日々の感情ログと月単位の成長物語が混在しやすい。GAM のアプローチで、session-level context（今日の日記）と organizational-level memory（人生の物語）を **明示的に 2層に分離** すれば、ナラティブ生成精度向上が期待できる。
