#### (2026-04-13, Cursor 3.1) Agents Window Tiled Layout Release

**URL**: https://cursor.com/changelog

Cursor 3.1 が April 13, 2026 に Agents Window の改善を実装。主な更新：

**キーポイント**:
- **Tiled Pane Layout**: 複数エージェントを並列管理する際、タイル状にペインを分割可能に。エージェント同士の出力を並べて比較が容易
- **Voice Input 改善**: Ctrl+M（またはCmd+M）で音声入力をサポート。波形表示、タイマー、キャンセルボタン付き
- **Persistent Session**: ペイン配置がセッション間で保存される

**focus-you への示唆**:
→ **HD運営のマルチエージェント・パイプラインの UI 設計参考**。現在の複数部署委譲（並列実行）では、各部署の進捗を **個別タブ/ウィンドウ** で管理しています。Cursor 3.1 の「タイル表示」パターンをダッシュボードに取り入れれば、社長が複数部署の進捗をリアルタイムで比較しながらハンドオフ判断できるようになる可能性。
